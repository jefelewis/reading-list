## Getting Started
**Prerequisites:**
* [Install Node.js](https://nodejs.org/en/download/)

**1. Install Dependencies:**
```
npm install
```

**2. Start Node.js Server:**
```
npm run server
```

**3. Start Webpack Development Server:**
```
npm start
```
